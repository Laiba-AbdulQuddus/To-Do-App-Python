# Todo App

A simple Python command-line Todo application to manage your daily tasks.

## Features
- Add tasks with name and status (Done/Pending)
- View all tasks
- Update task status
- Delete tasks
- Data stored persistently in a JSON file

## Technologies Used
- Python 3
- JSON for storage
